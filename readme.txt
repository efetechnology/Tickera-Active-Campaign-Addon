== Description ==

[Activecampaign](https://www.activecampaign.com/)

- you will use API Access in Settings of Developer (included in URL and Key) 
- you will use List ID in Lists from Domain.

### This is *the best* WordPress Activecampaign plugin.

### Integrate Active Campaign with Tickera
- Disable Active Campaign: This Checkbox will help you want to Enable or Disable Active Campaign.
- Status: The status for each list the contact is added to (active or unsubscribed)
- Instant Responders: Whether or not to set "do not send any future responders.
__Integrating with Salesforce has never been so simple.__

###Tickera + Active Campaign = A Powerful Combination

This free Active Campaign Add-On adds Entries lists from Tickera into Active Campaign automatically, making customer relationship management simple. The setup process takes a few seconds, and your Tickera contact form will be linked with Active Campaign.
